# 前言

欢迎来到本健康饮食APP的Gitee项目页面。本项目是一款以推广健康饮食习惯，帮助用户合理搭配饮食为宗旨的移动应用。这里，我们提供了完整的源码、文档报告及代码讲解，旨在帮助广大开发者学习交流，共同进步。

## 内容介绍

本项目基于Java语言开发，采用Spring Boot框架，结合前端技术JS、Vue、CSS3，构建出一款易于使用的健康饮食APP。用户可以通过此APP查询到各类食物的营养成分、搭配建议，还可以根据个人需求定制饮食计划。此外，APP还提供了丰富的健康饮食知识，让用户在享受美食的同时，也能更好地关注自己的身体健康。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、CSS3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.Js 12\14\16

## 核心代码

以下是项目中的一段核心代码，展示了如何使用Java查询数据库中的食物信息：

```java
@Autowired
private FoodRepository foodRepository;

public List<Food> findFoodsByName(String name) {
    return foodRepository.findByNameContaining(name);
}
```
